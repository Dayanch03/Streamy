# Streamy

<p align="center">
  <strong>Streamy</strong> — это легковесное и быстрое веб-приложение для мгновенного обмена файлами между устройствами в одной локальной сети (Wi-Fi). Никаких облачных серверов, флешек или сложных настроек — только локальная передача на максимальной скорости вашего роутера.
</p>

---

## 🚀 Ключевые особенности

* **P2P/Локальный стриминг:** Файлы передаются напрямую между устройствами внутри вашей Wi-Fi сети, не уходя в интернет. Это безопасно и конфиденциально.
* **Кроссплатформенность:** Работает на любом устройстве с браузером (iOS, Android, macOS, Windows, Linux) — достаточно просто отсканировать QR-код или ввести локальный IP.
* **Ультра-чистый UI/UX:** Минималистичный и дорогой интерфейс в стиле современных SaaS-платформ, поддерживающий drag-and-drop и микро-анимации.
* **Высокая скорость:** Скорость передачи ограничена только пропускной способностью вашей локальной Wi-Fi сети.

---

## 🛠 Технологический стек

Проект построен на базе современных и эффективных технологий, обеспечивающих быструю сборку и стабильную работу:

* **Frontend:** React, Tailwind CSS (для адаптивного и минималистичного дизайна).
* **Backend:** Node.js (или FastAPI / Flask) — для оркестрации комнат и локального стриминга данных.
* **Network:** WebSockets / HTTP-стримы для стабильной передачи чанков данных.

## 📦 Быстрый старт (Развертывание)

### Требования
Для запуска проекта вам понадобятся **Node.js** (версии 18+) и менеджер пакетов (npm, yarn или pnpm).

### 1. Клонирование репозитория
# Установка для бэкенда и фронтенда
npm install
3. Запуск в режиме разработки
Bash
npm run dev
После запуска консоль отобразит ваш локальный адрес в сети (например, http://192.168.1.45:3000).

💻 Как это работает?
Запуск: Вы запускаете сервер на своем основном устройстве (компьютере).

Подключение: Другие устройства (например, смартфон) подключаются к серверу, находясь в той же Wi-Fi сети.

Передача: Вы перетаскиваете файл в окно браузера, и он мгновенно становится доступен для скачивания на всех подключенных устройствах.

⚙️ Конфигурация Nginx (дляproduction-деплоя на домашний сервер)
Если вы хотите запустить Streamy на постоянной основе в своем домашнем серверном окружении (например, на Ubuntu Server), используйте следующий конфиг для Nginx:

Nginx
server {
    listen 80;
    server_name streamy.local; # или ваш локальный IP

    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
